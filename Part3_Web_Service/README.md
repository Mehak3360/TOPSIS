## How to Use

1. Open the web application:
   https://your-app-name.onrender.com

2. Upload the input CSV file.

3. Enter weights (comma separated).
   Example:
   1,1,1,2

4. Enter impacts (comma separated, use + or -).
   Example:
   +,+,-,+

5. Enter your email ID.

6. Click Submit.

7. The result file containing TOPSIS Score and Rank will be sent to the provided email.
